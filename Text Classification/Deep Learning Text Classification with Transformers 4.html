Deep Learning Text Classification with Transformers

Context

This project uses a BERT-based deep learning model to classify news articles into categories. The dataset contains recent Formula 1 news, and the objective is to build a robust model to predict the type of news article based on its content.

Objective

  • Build a multi-class text classification model using Transformers (BERT).
  • Preprocess textual data.
  • Tokenize with BERT tokenizer.
  • Train and evaluate the BERT-based model.

from transformers import TFBertForSequenceClassification, BertTokenizer
from transformers import create_optimizer

# Load model
model = TFBertForSequenceClassification.from_pretrained('bert-base-uncased', num_labels=len(labels))

# Use HF's built-in optimizer creator
batch_size = 16
num_train_steps = len(train_dataset) * 3  # epochs = 3
optimizer, schedule = create_optimizer(init_lr=5e-5,
                                       num_warmup_steps=0,
                                       num_train_steps=num_train_steps)

# Compile with optimizer and loss
loss = tf.keras.losses.SparseCategoricalCrossentropy(from_logits=True)
metrics = ['accuracy']

model.compile(optimizer=optimizer, loss=loss, metrics=metrics)
