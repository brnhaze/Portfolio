Liver Disease Deep Learning Image Classification 2

Context and Objective

This project builds a deep learning image classification model to detect liver disease from medical images. Using a custom CNN model, VGG16, and a third optimized model, we compare performance and adjust parameters to avoid Colab crashes (e.g., due to memory).

Project Context

Liver disease classification through histopathological images is essential for early detection and treatment. This project compares three deep learning models to classify liver diseases effectively.

Objective

To classify liver tissue images into multiple disease categories using:

  • Model 1: Custom CNN
  • Model 2: Transfer Learning using VGG16
  • Model 3: Optimized VGG16 with GlobalAveragePooling

Dataset Description

  • Images are organized into subfolders by class.
  • Training and testing sets are stored on Google Drive.
  • Images are resized to 128x128 for computational efficiency.

Conclusion and Recommendations

This project successfully explored deep learning-based classification of liver disease histology images using three different models:

  • Model 1: Custom CNN with 128128 images

  • Model 2: VGG16 transfer learning on 224224 images

  • Model 3: Improved CNN (deeper with Dropout and BatchNorm)

Key results:

  • Model 2 (VGG16) achieved the highest validation accuracy and generalization, but at the cost of higher memory and training time.

  • Model 1 was lightweight but showed underfitting on the validation set.

  • Model 3 balanced accuracy and efficiency, improving over Model 1.

  • Metrics like accuracy curves and confusion matrices showed Model 3 outperformed Model 1, and nearly matched Model 2.

Recommendation:

  • Use Model 3 in production for better efficiency and good accuracy, especially when resources are limited. If maximum accuracy is required and compute is not a constraint, go with Model 2 (VGG16).
